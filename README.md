# MyLang

Uma linguagem de programação escrita em C — este é o início da jornada!

## Como compilar

```bash
make
```

## Como executar

```bash
./mylang examples/hello.my
```

## Estrutura

- `src/`: código-fonte em C
- `include/`: headers
- `examples/`: programas na linguagem
- `tests/`: testes futuros
